Class Overview

This class represents an action plugin with the system. Please see com.newaer.sdk.plugins for documentation on how to add a plugin to the platform.

Constants

public static final String INTENT_DISCOVER

The action used to discover plugins. Your plugin should have an intent filter with this action and the default category in order to be discovered by the NewAer SDK.

Constant Value: "com.newaer.plugin.DISCOVER"

public static final String METADATA_NOTIFICATION_ICON

Metadata key in the manifest for the icon that will be displayed in the notification bar. Defaults to the application icon if none is set.

Constant Value: "newaer:notificationIcon"

public static final String METADATA_PLUGIN_EXPORTED

Metadata key in the manifest indicating if this plugin can be used by other applications using the NewAer SDK. Set to "false" to prevent other applications from calling your plugin. Default: true;

Constant Value: "newaer:exported"

public static final String METADATA_PLUGIN_NAME

Metadata key in the manifest for the name resource for a plugin. This will be used to display the plugin in user interfaces.

Constant Value: "newaer:pluginName"

public static final String METADATA_PLUGIN_RECEIVER

Metadata key in the manifest for the receiver class for a plugin. If this is not provided we expect a static inner class with the name ActionReceiver to receive the actions for this plugin.

Constant Value: "newaer:pluginAction"
