public abstract void configure (Context context)

Launches the configuration for this action.

This will start a new activity with the plugin's configuration UI. If the user does not finish configuration of the plugin in that activity then isConfigured() will return false.

Note that an action can not be fired, even if it is part of rule, until it has been configured. Developers should reflect this in the user interface of their applications.

Parameters
context The context to work in.
