BasePluginNoConfigRequiredActivity
BasePluginNoConfigRequiredActivity This class makes it easy to write a plugin which does not require any configuration. 

Class Overview

This class should be extended by plugins which require a user interface in order to configure them. See NAActionPlugin for various metadata you can and must specify in the manifest for your activity.

Subclasses must call setResultAndFinish(int) when they are finished. If you call finish() instead a RuntimeException will be thrown.

It is suggested that plugins store their configuration as json or xml but they are free to store anything that can be converted to a String representation.

Plugins always consist of an Activity and a BroadcastReceiver which must extend classes in this package.
